# SISY.work - Career and Opportunities Board for Women Experts

Live site: https://sisy.work

Part of the [SISY ecosystem](https://sisy.world) - visibility infrastructure for women experts.

## About

Advisory roles, board seats, consulting gigs, and speaking opportunities - curated for women experts. Includes a reverse job board where experts list availability and companies come to them.
